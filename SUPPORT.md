# Support Anime Content Engine

If this project helps you, there are several easy ways to support it.

# Star the Repository

Starring the repository helps more developers discover the project through GitHub search, topic pages, and recommendations.

# Share the Project

You can help by sharing the repository with:

- Anime developers
- Bot builders
- Social media automation communities
- Node.js developers
- People building Bluesky or X automation projects

# Open Useful Issues

Helpful issues include:

- Bug reports
- Workflow reliability issues
- Accessibility improvements
- Better filtering ideas
- Documentation gaps
- Platform integration ideas

# Contribute Code

Code contributions are welcome.

Please remember:

- Open pull requests against `develop`
- Do not open pull requests directly against `main`

Contribution details:

- [CONTRIBUTING.md](./CONTRIBUTING.md)

# Sponsor the Work

If you want to support the project financially, you can sponsor the work through GitHub Sponsors:

- https://github.com/sponsors/jaganganesh

The repository already includes GitHub funding configuration, so sponsorship links are surfaced directly on GitHub when available.

# Other Helpful Support

You can also help by:

- Watching the repository for updates
- Forking the project
- Sending improvement suggestions
- Referencing the project in articles, videos, or tutorials

Every star, share, issue, and contribution helps the project grow.
